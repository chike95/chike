## MVC

![alt text](img/image-2.png)

- Model：数据模型，一般用来存储数据，操作数据
- View：视图，一般用来展示数据
- Controller：控制器，一般用来处理数据，操作数据

## MVVM

![alt text](img/image-3.png)

- Model：数据模型，一般用来存储数据，操作数据
- View：视图，一般用来展示数据
- ViewModel：视图模型，用来连接Model和View，一般用来处理数据，操作数据
